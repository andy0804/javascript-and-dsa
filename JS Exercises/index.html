Centering an image

Center Vertically - Using padding There are many ways to center an element vertically in CSS. A simple solution is to use top and bottom padding:

.center { padding: 70px 0; border: 3px solid green; }

Flexbox Centering

A container with both the justify-content and the align-items properties set to center will align the item(s) in the center (in both axis).

.flexCenter { display: flex; justify-content: center; align-items: center; height: 200px; border: 3px solid green; }
