Algorithm


1. IF !root
   return

1.A DEFINE EMPTY ARRAY RESULTS = []

2. DEFINE A FUNCTION NAMED LOT (ROOT,LEVEL)
  2.1 IF !ROOT
      RETURN
  2.2 IF RESULTS[LEVEL] IS DEFINED
         RESULTS[LEVEL].PUSH VALUE
      ELSE
      RESULTS[LEVEL] = VALUE

3. CALL LOT(ROOT.LEFT ,LEVEL+1)
4. CALL LOT(ROOT.RIGHT, LEVEL + 1)
5. RETURN RESULTS.MAP(ELEMENT,INDEXT)=> ELEMENT IS EVEN ? REVERSE : RETURN ELEMENT
